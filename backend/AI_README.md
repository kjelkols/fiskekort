# AI_README.md

Dette dokumentet skal alltid leses av AI før oppstart av en oppgave i dette prosjektet.

## Viktige retningslinjer

- Bruk alltid `uv` (uvicorn) som server for FastAPI i Python.
- Følg prosjektets katalogstruktur og plasser alle dokumenter under docs/ eller underkataloger.
- Oppdater og bruk requirements.txt for alle Python-avhengigheter.
- Bruk Alembic for database-migreringer.
- Følg datamodell og API-spesifikasjon for alle endringer.
- Skriv og oppdater tester for både backend og frontend.
- Dokumenter alle endringer og nye funksjoner i relevante md-filer.
- Sjekk alltid om det finnes oppdaterte retningslinjer i denne filen før du starter en ny oppgave.
